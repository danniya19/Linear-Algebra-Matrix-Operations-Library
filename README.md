# Matrix Operations Tool

A web-based utility for performing fundamental linear algebra operations. This tool allows users to dynamically generate matrices and calculate results for multiple matrix operations using custom JavaScript algorithms.

## Features

- **Dynamic Matrix Generation**: Users can specify the number of rows and columns to generate input fields in real-time.
- **Multiple Matrix Support**: Ability to perform operations on multiple matrices (Addition, Subtraction, Multiplication).
- **Core Operations**:
    - **Addition & Subtraction**: Element-wise calculation for any number of matrices.
    - **Multiplication**: Standard matrix multiplication algorithm.
    - **Determinant**: Recursive calculation for square matrices.
    - **Inverse**: Matrix inversion for 2x2 matrices.
- **Responsive Interface**: Built with a clean, centered UI for better user experience.

## Tech Stack

- **HTML5**: Structured input forms and dynamic containers.
- **CSS3**: Custom styling for a modern look and feel.
- **JavaScript (ES6)**: Core logic for matrix mathematics and DOM manipulation.

## How to Use

1. Select the desired operation from the dropdown menu.
2. Define the dimensions (Rows and Columns) for your matrices.
3. Specify how many matrices you want to include in the operation.
4. Fill in the generated input fields with your numeric values.
5. Click **Calculate** to see the result displayed in the result box.
